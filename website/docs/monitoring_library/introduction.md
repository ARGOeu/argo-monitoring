## The ARGO Monitoring Service library

Tne ARGO Monitoring Service library is a simple Python library for interacting with the ARGO Monitoring Service REST API. 
It intends to provide access to basic functionality of the API, in a simple and intuitive way.
The source code is available at the [ARGOeu repository](https://github.com/ARGOeu/argo-mon-library) on Github, under the Apache 2.0 license.

In the following sections, you may find information on installation methods, authentication, and a few usage examples.
